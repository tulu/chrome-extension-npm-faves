npm faves

Thank you for using npm faves!

Follow the steps to get started.

Add packages

Start your amazing experience by adding some packages.

Navigate to https://www.npmjs.com/, search the package and press the "Add to faves" button next to the package name.

The package will be added to your faves list in the extension.

View packages

After adding some packages you can find them in the extension pop up

Open the extension from the toolbar and the list of faves will be available to navigate.

If the extension is not visible pin it to the toolbar.

Copy install snippet

After adding a package you can find it in the pop up and copy the install snippet.

Open the extension from the toolbar and navigate to the package.

Click on the install snippet and it will be copied to the clipboard

npm faves is an open source extension from devs to devs
